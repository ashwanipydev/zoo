# Design System: The Botanical Archive

## 1. Overview & Creative North Star
This design system is built to transform a standard administrative interface into a high-end editorial experience. Moving away from the cluttered "control room" aesthetic, we adopt the **Creative North Star: The Botanical Archive**.

The goal is to evoke the feeling of a curated scientific journal—premium, authoritative, yet deeply connected to the natural world.

## 3. Typography: The Editorial Voice
*   **Headline/Display**: Manrope (Geometric Clarity)
*   **Body**: Manrope (1rem / 1.6 Line Height)
*   **Labels**: Inter (Technical Metadata feel)

## 4. Components & Principles
*   **No-Line Rule**: Prohibit 1px solid borders. Use tonal shifts for sectioning.
*   **Glassmorphism**: Use for floating elements (80% opacity + 20px blur).
*   **Tactile Buttons**: `xl` roundedness (1.5rem).
*   **Pebble Chips**: Full roundedness for status indicators.
